Timestamp API Instructions

This API works with UTC time.

You can pass unix timestamp or a date string to get back a JSON containing the unix timestamp and a date string.

The acceptable string is formated like "Month(string), Day(number) Year(Four Digits)". Ex: December 15, 2015

Simply pass a string or unix timestamp on your URL to test.

Examples:

  • December 15, 2015
  • 1450137600 (Unix timestamp)
